# Sber2Excel

Конвертер PDF-выписок дебетовых карт Сбербанка в CSV и Excel.

## Что делает

Загружает PDF-выписку, скачанную из СберБанк Онлайн (раздел «Выписки и справки»), разбирает все операции и позволяет сохранить результат в:

- **CSV** — с UTF-8 BOM, разделитель `;`, удобно для открытия в Excel
- **XLSX** — два листа: «Информация» (реквизиты счёта, итоги) и «Операции» (таблица транзакций с цветовым выделением поступлений и списаний)

## Скриншот

После загрузки PDF отображается:
- шапка с владельцем счёта, периодом, пополнением и списанием
- таблица операций с сортировкой по любой колонке
- строка статуса с количеством операций

## Требования

- Windows 10 / 11
- [.NET 10 Runtime](https://dotnet.microsoft.com/download/dotnet/10.0)

## Запуск из исходников

```bash
git clone <repo>
cd Sber2Excel
dotnet run
```

## Сборка релиза

Рантайм .NET встроен в бинарник, PDB не генерируются.

**Windows:**
```bash
dotnet publish -c Release -r win-x64
```
Результат: `bin/Release/net10.0/win-x64/publish/Sber2Excel.exe` + 3 нативных `.dll`

**Linux x64:**
```bash
dotnet publish -c Release -r linux-x64
```

**Linux ARM64** (Raspberry Pi 4, AWS Graviton и т.п.):
```bash
dotnet publish -c Release -r linux-arm64
```

Кросс-компиляция поддерживается — Linux-бинарник можно собрать из Windows и наоборот.

## Поддерживаемый формат

Выписки по дебетовым картам Сбербанка, экспортированные из **СберБанк Онлайн** в PDF.  
Формат: многостраничный документ с таблицей операций следующей структуры:

| Дата операции (МСК) | Категория | Сумма в валюте счёта | Остаток |
|---|---|---|---|
| Дата обработки + код авт. | Описание операции | — | — |

Протестировано на выписках за 2025 год (формат страницы A4, до 20 страниц).

## Колонки в экспорте

| Колонка | Описание |
|---|---|
| Дата операции | Дата и время по МСК |
| Дата обработки | Дата проведения по счёту |
| Код авторизации | 5–6-значный код банка |
| Категория | Тип операции (Перевод СБП, Прочие операции и т.д.) |
| Описание | Детали: получатель/отправитель, источник операции |
| Сумма | Положительная — поступление, отрицательная — списание |
| Остаток | Остаток на счёте после операции |

## Стек

| Компонент | Библиотека |
|---|---|
| UI | [Avalonia 12](https://avaloniaui.net/) + CommunityToolkit.Mvvm |
| Парсинг PDF | [PdfPig 0.1.14](https://github.com/UglyToad/PdfPig) |
| Экспорт Excel | [ClosedXML 0.104.2](https://github.com/ClosedXML/ClosedXML) |
| Экспорт CSV | Встроенный `StreamWriter` (UTF-8 BOM) |

## Ограничения

- Поддерживается только рублёвый счёт (операции в иностранной валюте отображаются корректно, но валюта операции не экспортируется).
- Выписки с другим форматом (например, кредитные карты Сбера или другие банки) не поддерживаются.
